Hello, All This is the readme file for helping the user to execute the code in following ways:
For Python Notebook
1 Install the Jupyter Notebook from the Anaconda Website.
2 create a Directory on Local Machine.
3 Open the given Notebook with the extension of "ipynb".

For Flask Code running locally
1 - Install the libraries which is given in Requirement.txt file.
2 - open the CMD prompt in Local machine and install all the libraries with 'pip'command.
3 - install the VS code and put the template and app.py file into the working directory.
4 run the app.py file , the application will deploy on the local machine.
